# BUDGET

harvest-engineの年間予算・費目別配分・支出台帳・PL計画を管理する。
本ドキュメントの変更はオーナー専決事項である([[GOVERNANCE]]参照)。

|---|---|
| 需要テスト | 80,000円 |
| API | 48,000円 |
| 法務 | 40,000円 |
| ドメイン・雑費 | 12,000円 |
| 予備 | 20,000円 |
| **合計** | **200,000円** |

## 経理ルール

- 費目残額が予算の20%を切った時点で、執行役(GM)がHANDOFF.mdに警報を記載する
- 費目間の流用は予備費のみ執行役決裁で可。それ以外の費目間の組み替えはオーナー決裁が必要
